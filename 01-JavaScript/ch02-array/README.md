## Ch02. 배열

### 1.배열
- 기본 자료형
- var jina = ['jana', 'woman', 32, 'Korea']

|   0  |   1   |   2  |   3   |  
| :--: | :---: | :--: | :---: |  
| jana | woman |  32  | Korea |  

```javascript
var a = ['hi', 234, '434', 'you'];

a[0]; // "hi"
a[3]; // "you"

// 배열추가
a[4] = 98;

// 배열을 추가한 후
a = ['hi', 234, '434', 'you', 98];

// 배열의 길이
a.lenght // 5

a.indexof(234) //2
```
